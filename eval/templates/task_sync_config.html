Configuration Format

Please upload a config.json file with the following structure: 

{
  "project_code": "Code-Data-Repack-SFT",
  "sheet_url": "https://docs.google.com/spreadsheets/d/...",
  "sync_interval_minutes": 60,
  "primary_key_column": "question_id",
  "scraping_needed": false,
  "link_column": "problem_link",
  "column_mapping": {
    "prompt": "Task Prompt",
    "reference_links": "Links"
  },
  "sync_mode": "prompt_in_sheet",
  "sheet_tab": "Sheet1",
  "is_active": true
}
  • project_code: The code of the project to apply this config to.
  • sheet_url: Google Sheet or data source URL.
  • sync_interval_minutes: How often to sync (in minutes).
  • primary_key_column: Unique identifier column for tasks.
  • scraping_needed: Set to true if scraping is required.
  • link_column: Column containing links for scraping (if needed).
  • column_mapping: Map logical fields to sheet columns.
  • sync_mode: One of "prompt_in_sheet", "scraping", or "custom".
  • sheet_tab: Sheet tab/range name (optional).
  • is_active: Set to true to activate this config.
